# syntax marathon

syntax marathon is a modern typing speed game built using HTML, CSS, and JavaScript.

The game features a futuristic purple-themed UI with real-time typing validation, WPM tracking, accuracy calculation, difficulty modes, and score system.

---

# 🚀 Features

✅ Real-time typing validation  
✅ Live WPM counter  
✅ Accuracy tracking  
✅ Difficulty levels  
✅ Responsive UI  
✅ Animated background  
✅ Score system  
✅ High score saving using LocalStorage  
✅ Smooth animations and hover effects  

---

# 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript

---

# 🎯 Game Modes

- Easy
- Medium
- Hard

Each mode includes different coding snippets and typing challenges.


3. Install the Live Server extension

```text
index.html
```

5. Click:

```text
Open with Live Server
```
